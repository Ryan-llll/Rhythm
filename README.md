# Rhythm — Habit Tracker

Rhythm is a clean, developer-centric habit tracker featuring a premium dark mode, GitHub-style contribution heatmaps, streaks, and a monthly logger.

## 🛠️ Features

1. **Check Off Habits**: Tap checkboxes in the weekly dashboard.
2. **Multi-completions**: Hover over a completed checkmark to increment count (e.g. `x2`, `x3`).
3. **Multi-slot habits**: For habits like Prayers, click the slot fraction button to open individual slot toggles.
4. **View Details**: Click on a habit's name to open its modal, where you can view its **6-Month Heatmap**, navigate the **Monthly Calendar** (and click past days to backfill entries), or **Edit/Delete** it.
5. **Backups**: Use the sidebar links to export or import your data as a backup JSON file.
